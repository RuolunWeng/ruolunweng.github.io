About Project

"sfPivoine" is a musical application allowing everybody to be engaged in a participatory concert in which smartphone is the main musical instrument generating sound with gestures. This is a performance for Pipa, Smartphones and Live electronic music, which is inspired by an Chinese opera "The Peony Pavilion"(牡丹亭).
Everybody can get the feeling of being a part of performance and part of scenario.
